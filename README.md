# BremaAI
A professional AI Chat Application built with Flutter on Ubuntu 24.04.

## ✨ Features
* **Powered by Gemini 2.5 Flash-Lite**: High-speed, high-quota AI responses.
* **Beautiful Markdown**: Supports code highlighting for Python, JS, and more.
* **Ubuntu Native**: Optimized for the Linux Desktop with a custom sidebar icon.
* **Selectable Text**: Easily copy AI responses and code snippets.

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/bundlab/brema_ai.git`
2. Get dependencies: `flutter pub get`
3. Run the app: 
   `flutter run -d linux --dart-define=API_KEY=YOUR_KEY_HERE`

## 🛠 Tech Stack
* **Frontend**: Flutter (Dart)
* **AI Engine**: Google Generative AI (Gemini)
* **OS**: Ubuntu 24.04 LTS
